# Версия
Это утилит семантического управления версиями. Её можно использовать для хранения объектов управления версиями для флагов функций. Также её можно использовать вместе с модулем флагов для создания профессионально оформленного флага версии.

## Использование

### Конструкция
Множество способов создания объекта версии, поскольку множество перегрузок конструктора. Для печати доступна перегрузка `operator<<`.

### Модификация
Чтобы становить любого свойства объекта версии каждому свойству назначен определенный сеттер.
Для изменения любого свойства объекта доступны следующие функции:
- Release: Увеличить основной номер объекта и установить его дополнительный номер и номер версии исправления на 0.
- Update: Увеличить дополнительный номер объекта и установить версию исправления объекта на 0, не изменяя основной номер объекта.
- Patch: Увеличить значение исправления объекта, не изменяя основной или дополнительный номера объекта.
- Publish: Установить основной номер объекта на 1. Если объект оценивается как уже опубликованный, возникает ошибка, и никакие значения не изменяются.

### Сравнивать
Каждое числовое свойство объекта имеет функцию сравнения. Для сравнения одного объекта версии с другим доступна функция `compare`. Доступна функция `is_public`, используемая в `publish`.
