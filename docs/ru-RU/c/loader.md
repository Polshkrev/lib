## Использование
### Загрузить
Чтобы загрузить библиотеки следует использовать функцию `library_load`. Эта функция вернет дескриптор динамически подключаемой библиотеки.
### Доступить
Чтобы доступить к функции внутри динамически подключаемой библиотеки необходимо вызвать функцию `library_function`.
Эта функция принимает дескриптор библиотеки, полученный с помощью функции `library_load`, и имя загружаемой функции. Функция вернет дескриптор загруженной функции.
### Деструкция
Чтобы освобожденить памяти библиотеки необходимо вызвать функцию `library_delete`. Она вернет логическое значение, указывающее на успешный результат.
