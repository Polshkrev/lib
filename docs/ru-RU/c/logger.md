## Использование
### Конструкция
Чтобы создать новый объект логгера, используйте функцию `logger_new`. Эта функция вернёт объект логгера, выделенный в куче. Параметрами функции являются имя и значение перечисления уровня логирования по умолчанию. Как и в моей [оболочке](https://github.com/Polshkrev/Utilities/tree/main/docs/en-UK/globals/log) вокруг пакета логирования Python, параметр уровня, передаваемый этой функции, задаёт минимальный уровень вывода логгера.
Для правильной настройки логгера необходимо добавить `stdout` или файл. Для этого необходимо вызвать функцию `logger_add_console` или `logger_add_file` соответственно. Чтобы использовать и `stdout`, и файл, необходимо вызвать функцию `logger_full_setup`.
### Логирование
Чтобы записать сообщение в журнал, необходимо вызвать функцию `logger_log`. Эта функция получает представление объекта logger, сообщение журнала, завершающееся нулем без символа новой строки, и уровень, на котором сообщение будет записано в журнал. Чтобы изменить уровень журналирования по умолчанию, вызовите функцию `logger_set_level`.
### Деструкция
Чтобы освободить объект регистратора, вызовите функцию `close_logger`.
