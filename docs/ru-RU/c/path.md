## Использование
### Конструкция
Чтобы создать нового объекта адреса можно использовать две функции: `path_init` и `path_from`. Как правило, любая функция, возвращающая `path_t`, может считаться возвращающей новый адрес, выделенный в куче.
### Модификация
Чтобы изменить адреса определены три функции: `path_append`, которая объединяет два заданных адреса, `path_append_as`, которая добавляет строку C к адреса, и `path_append_to`, которая добавляет адрес к строке C.
### Деструкция
В связи с тем, что адрес выделяется в куче, для освобождения этого адреса необходимо вызвать функцию `path_delete`.
