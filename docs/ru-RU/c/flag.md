## Использование

### Конструкция
В настоящий момент поддерживаются только следующие типы флагов: логическое значение (boolean), строка (string) и 64-битное беззнаковое целое число (unsigned integer). Для создания флага любого из этих типов используйте функции `flag_string`, `flag_bool` или `flag_uint64`. Каждый из методов принимает имя флага &mdash; например, `o`, `help` &mdash; значение по умолчанию, которое может быть null, и описание для справочного сообщения. Эти функции возвращают указатель на строку, логическое значение или 64-битное беззнаковое целое число в статической памяти.

### Доступить
Перед использованием значения флага необходимо вызвать функцию `flag_parse` после того, как вы сформируете каждый флаг. Эта функция принимает значения `argc` и `argv` из основной функции. Затем вы можете использовать каждое из установленных значений флага. Чтобы задать имя вашей программы в справочном сообщении, вы можете использовать функцию `flag_set_programme_name`.

#### Диапазон
Вы можете установить минимальное и максимальное допустимые значения для флага типа uint64, вызвав функцию `flag_uint64_range`. Эта функция установит минимальное и максимальное значение флага, которое будет выведено в справочное сообщение. Если значение, переданное флагу во время выполнения, меньше минимального значения или больше максимального значения, будет сгенерирована ошибка, которая будет выведена в `stderr`.
